Go-based WhatsApp activity tracker bot using whatsmeow and SQLite. Main entrypoint is cmd/bot/main.go. Architecture is layered: internal/domain defines entities and repository interfaces, internal/app/usecase contains business logic for commands like #lapor, #leaderboard, #mystats, achievements, Strava webhook handling, and session reset, internal/infra contains SQLite repository, WhatsApp integration, Strava client, and HTTP handlers. The bot focuses on one configured WhatsApp group and persists user activity/streak data in SQLite.

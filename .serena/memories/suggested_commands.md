Useful commands for this repo on Darwin:
- `go test ./...` — run the Go test suite
- `gofmt -w <files>` — format edited Go files
- `go run ./cmd/bot/main.go` — run the bot in development
- `go build -o bot ./cmd/bot/main.go` — build the bot binary
- `rg -n "pattern" internal -g '*.go'` — fast code search
- `rg --files` — list tracked files quickly
